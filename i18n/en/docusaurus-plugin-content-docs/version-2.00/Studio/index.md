## Introduction

Daiana Studio is a visual automation platform designed to build intelligent workflows using configurable nodes and artificial intelligence agents.

Daiana Studio allows technical and non-technical users to create, customize, and scale conversational processes, automated analytics, API integrations, and complex operations without writing code.
