The **Reports** section allows you to generate detailed reports on platform usage based on different filters. This tool is useful for performance analysis, user tracking, and continuous improvement of the conversational experience.
<Banerimg img="dashboard/reportes.png" />


## Available Filters

You can adjust the following parameters before generating the report:

1.  **Report Type** (required)
2.  **Start and End Date**
3.  **Team**
4.  **User**
5.  **Agent**

Once the desired criteria are selected, click **"Generate Report"** to obtain the results.
